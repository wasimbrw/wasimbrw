## 🔧 Deployment

### GitHub
1. Create a new GitHub repo
2. Push all files: `git push`

### Vercel
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repo
3. Deploy — done! ✅

No build step needed. Static HTML site deploys instantly.
